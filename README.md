##目录结构

  config 存放默认文件
  
  lib 存放操作数据库文件
  
  middlewares 存放判断登录与否文件
  
  public 存放样式和头像文件
  
  routes 存放路由文件
  
  views 存放模板文件
  
  index 程序主文件
  
  package.json 包括项目名、作者、依赖等等
  
##各模块用处
  koa node框架
  
  koa-bodyparser 表单解析中间件
  
  koa-mysql-session、koa-session-minimal 处理数据库的中间件
  
  koa-router 路由中间件
  
  koa-static 静态资源加载中间件
  
  ejs 模板引擎
  
  md5 密码加密
  
  moment 时间中间件
  
  mysql 数据库
  
  markdown-it markdown语法
  
  koa-views 模板呈现中间件
  
  chai mocha 测试使用
  
  koa-static-cache 文件缓存
